MSound - The Matlab Sound
ReadMe File
-------------------------
Per poter utilizzare al meglio i listati presenti nelle sottocartelle si consiglia
di ricopiare l'intera cartella sul proprio disco locale, quindi aggiornare le
pathlist di Matlab. In particolare, è necessario registrare le directory:

- czerny : file di esempio (Carl Czerny - Studio Op. 740 Num. 14)
- esempi : file di esempio
- msound : routine indispensabili
- note : scala cromatica temperata
- noteconst : definizione delle note secondo la scala cromatica temperata (costanti)
- notefunc : definizione delle note secondo la scala cromatica temperata (funzioni)
